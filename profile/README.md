# Max — Audio Converter with CD Ripping (Paid Support)

## About Max

Max provides audio conversion and CD ripping with paid support for Mac users, audio archivists, and music collectors. Max combines lossless and lossy encoding, MusicBrainz tagging, and batch processing. Max supports FLAC, MP3, ALAC, WAV, Ogg, and output device selection. Max enables users to extract audio CDs, transcode between formats, and preserve metadata. Max is trusted by radio producers, podcast editors, music librarians, and home archivers worldwide.

## Deep Dive — Audio Conversion Features
CD ripping with MusicBrainz database lookup for track metadata
Format conversion between FLAC, ALAC, WAV, MP3, AAC, Ogg, WMA
Batch processing of entire folders with recursive scanning
Output format presets for portable devices (iPhone, iPod, Android)
Error detection and frame-level quality reporting for discs
